# BestGames
BestGames — это маркетплейс для поиска и покупки компьютерных игр и внутриигровых предметов. Сервис объединяет:
- разработчиков компьютерных игр
- авторские сообщества
- геймеров
- инвесторов
- рекламные компании

## Целевая аудитория
- молодые люди в возрасте 18-35 лет
- компании, принимающие участие в кибер-турнирах
